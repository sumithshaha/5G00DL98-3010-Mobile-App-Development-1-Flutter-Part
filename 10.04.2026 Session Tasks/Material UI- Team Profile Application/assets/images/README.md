# assets/images

Drop your team-member photos here. The default `team_data.dart` expects:

- `sumith.jpg`
- `member2.jpg`
- `member3.jpg`

Any JPG or PNG works. If a file is missing the MemberCard widget falls back
to a generic avatar icon, so the app still runs while you're gathering the
real photos.

For the homework submission, replace these placeholder entries with photos
of your actual teammates and update the paths in `lib/data/team_data.dart`.
