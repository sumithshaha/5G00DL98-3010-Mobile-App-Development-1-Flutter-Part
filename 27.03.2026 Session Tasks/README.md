# Homework 1: Simple Login Screen

A Flutter UI-only login screen created as part of the Widgets and UI Construction homework.

## Setup

1. Create a new Flutter project: `flutter create login_screen`
2. Replace `lib/main.dart` with the provided `main.dart`
3. Replace `pubspec.yaml` with the provided `pubspec.yaml`
4. Create the folder `assets/images/` in your project root
5. Place any logo image as `assets/images/logo.png` (e.g., your university logo)
6. Run `flutter pub get`
7. Run with `flutter run`

## What's included

- Logo image loaded from assets
- Username text field with icon
- Password text field (obscured) with icon
- "Remember me" checkbox (functional toggle)
- Login button (no backend logic)
- Clean layout using Column, SizedBox, Padding, Row, and Center
